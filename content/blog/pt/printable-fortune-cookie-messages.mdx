## Como Usar Esses Bilhetes de Biscoito da Sorte para Imprimir (Guia Rápido)

1. Copie um conjunto de mensagens abaixo para um documento (Google Docs / Word / Canva).
2. Formate como tiras pequenas (busque **1 linha** por bilhete).
3. Imprima em papel comum ou papel cartão leve.
4. Recorte as tiras e dobre uma vez.
5. Insira em biscoitos caseiros, mini envelopes ou sacolinhas de lembrancinha.

<Callout type="tip" title="Dica de impressão">
Faça um teste rápido antes. Se o texto quebrar a linha, diminua o tamanho da fonte ou encurte a frase — os bilhetes funcionam melhor quando lidos de uma só vez.
</Callout>

### Modelo 1: 60 bilhetes curtos para imprimir

“Escreva 60 mensagens de biscoito da sorte para **[evento]**. Tom: **[engraçado/doce/neutro]**. Uma frase cada, com menos de 85 caracteres. Sem números da sorte.”

### Modelo 2: Conjunto para sala de aula (comportamento + incentivo)

“Escreva 30 bilhetes de biscoito da sorte para sala de aula para **[nível escolar]**. Linguagem simples. Foque em esforço, bondade, curiosidade. Uma frase cada.”

### Modelo 3: Conjunto de valorização no trabalho (por comportamento)

## Perguntas Frequentes

### Posso usar essas mensagens comercialmente?

Se você está imprimindo para um evento privado (festa/sala de aula/equipe), geralmente está tudo bem. Se for vender produtos, considere gerar um conjunto original no [/generator](/generator) para que seus bilhetes sejam exclusivos.

### Devo incluir números da sorte?

Pode incluir, mas a maioria dos bilhetes modernos fica mais limpa sem eles. Se for adicionar números, mantenha a mensagem curta e fácil de ler.

## How to Use These Printable Fortune Slips (Quick Guide)

1. Copy a set of messages below into a doc (Google Docs / Word / Canva).
2. Format as small strips (aim for **1 line** per slip).
3. Print on regular paper or light cardstock.
4. Cut the strips and fold once.
5. Insert into homemade cookies, mini envelopes, or party favor bags.

<Callout type="tip" title="Print tip">
Do a quick test sheet first. If your text wraps, reduce font size or shorten the line—fortune slips feel best when they read in one breath.
</Callout>

## FAQ

### Can I use these messages commercially?

If you’re printing for a private event (party/classroom/team), you’re typically fine. If you’re selling products, consider generating an original set at [/generator](/generator) so your slips are unique.

### Should I include lucky numbers?

You can, but most modern slips feel cleaner without them. If you do add numbers, keep the message itself short and readable.
